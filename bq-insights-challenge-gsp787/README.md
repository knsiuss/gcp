# Derive Insights from BigQuery Data: Challenge Lab (GSP787) - Task 5 Solver

Automated script for GSP787 Task 5. This script automates running the BigQuery query to identify the day when cumulative deaths in Italy crossed 10,000.

## Quick Start

1. Start the lab in Qwiklabs.
2. Open **Cloud Shell** and run:

```bash
git clone https://github.com/knsiuss/gcp.git gcp-labs
cd gcp-labs/bq-insights-challenge-gsp787
chmod +x solve_task5.sh
./solve_task5.sh
```

3. Once the query completes, click the "Check my progress" button for Task 5 ("Identify a specific day") to get points!
