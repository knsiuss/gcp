# Working with JSON, Arrays, and Structs in BigQuery (GSP416)

Automated script for GSP416. This script automates creating the datasets (`fruit_store`, `racing`), loading newline-delimited JSON data from Cloud Storage using autodetect and custom nested schemas, running arrays aggregation, page/product lengths, nested unnesting, struct records, and array operations in BigQuery.

## Quick Start

1. Start the lab in Qwiklabs.
2. Open **Cloud Shell** and run:

```bash
git clone https://github.com/knsiuss/gcp.git gcp-labs
cd gcp-labs/bq-json-arrays-structs
chmod +x solve_json_arrays.sh
./solve_json_arrays.sh
```

3. The script will execute all tasks. Once done, you can click all "Check my progress" buttons to get **100/100**!
